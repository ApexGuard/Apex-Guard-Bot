### Apex Guard Project
- Powerd by apexguard, openrouter.ai (ai detection), virus total (malicious files & urls detection) & NSFWJS (nsfw detection)
- ApexGuard is a Discord bot that helps you protect your server from various types of abuse and harassment.
- It can detect and block unsafe content, including hate speech, harassment, and other forms of abuse.

# Why ApexGuard?
- Our bad words list updates and have more then 1000 words in diffrent languages.
- Our nsfw system is optmized so it can detect nsfw content with high accuracy even if the attatchment is gif / video.
- Our anti scam system is optmized so it can detect scam with high accuracy by many ways : regex, ai, images hash, etc.
- We are the first opensource project that provided anti webhook exploit system, so you can protect your server from webhook spam exploit.
- Our Project is completly non-profit and open source to help the community.
- ApexGuard was founded after a lot of research and testing to provide the best protection for your server.

# How to setup
- Clone the repo :
- git clone https://github.com/apexguard/apexguard.git

- Install dependencies :
- npm install

- Configure the bot :
- Copy the .env.example file to .env
- Fill in the values in the .env file

- Load the commands (one time) :
- node deploy-commands.js

- Run the bot :
- node index.js
